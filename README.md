---

## Dependencies

1. [Go Programming Language][1]
2. `go get -u golang.org/x/vgo` (until Go 1.11 comes out ~ August 2018)

[1]: https://golang.org

## Running

The easiest way of running for development purposes is

```sh
vgo run cheapskate.go --logtostderr --plugins.disabled=""
```
